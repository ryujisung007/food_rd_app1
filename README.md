# 🧪 Food R&D Platform

식품 연구개발 통합 플랫폼 — 배합비 설계부터 원가·공정·표시사항까지

## 핵심 기능 5가지

| # | 기능 | 설명 |
|---|------|------|
| 1 | **💰 원가 분석** | 원재료 단가 DB(36종) → 배합비 기반 원가표 자동 생성 |
| 2 | **⚗️ 배합비 설계** | 100% 기준(백분율) 배합표, AI 자동생성 또는 직접 CSV 입력 |
| 3 | **🔀 표준배합비 비교** | 5종 표준배합비 vs 내 배합비 비교 분석 (차이 차트) |
| 4 | **🔬 AI 공정 분석** | PDF 업로드 → AI 연구원이 학습 후 공정 리스크 분석 |
| 5 | **🏷️ 표시사항 검토** | 식품등의 표시기준 PDF 학습 → 적합성 비교표 |

## 전체 페이지 (10개)

1. 📈 매출추이 — 음료 카테고리 매출 트렌드
2. 🏷️ 브랜드분석 — 브랜드별 매출, 점유율
3. 🤖 AI제품카드 — 9종 제품 → AI 배합비 생성
4. ⚗️ 배합비설계 — 100% 배합표 + 표준비교 + 원가연동
5. 🏭 공정리스크 — 8단계 공정, HACCP CCP
6. 📋 규제서류 — 품목제조보고서
7. ✏️ 배합연습 — CSV 작성 + 검증 + 표준비교 + 저장
8. 💰 원가분석 — 원재료 단가 DB + 원가계산
9. 🔬 AI공정분석 — PDF 학습 + AI 리스크 분석
10. 🏷️ 표시사항 — 표시기준 PDF 학습 + 적합성 비교

## 설치 & 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Streamlit Cloud 배포

1. GitHub 레포 루트에 모든 파일 배치
2. [share.streamlit.io](https://share.streamlit.io) → New app
3. Main file path: `app.py`
